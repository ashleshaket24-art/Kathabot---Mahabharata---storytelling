# Kathabot – Mahabharata Storytelling Chatbot

## Description
Kathabot is an AI-based chatbot designed to narrate stories from the Mahabharata in a simple and interactive way for school students.

## Features
- Storytelling in conversational style
- No login or registration
- Character and event-based stories

## Technologies Used
- HTML, CSS, JavaScript
- Python / Java
- Basic NLP / AI logic

## Project Flow
User → Chat Interface → AI Processing → Story Response
